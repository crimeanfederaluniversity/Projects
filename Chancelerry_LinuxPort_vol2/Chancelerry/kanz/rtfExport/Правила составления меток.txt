Макет метки V2
Общий шаблон
#mark(n)&S...&E&S...&E#
n - параметр метки это инстанса
	0 - первые найденные данные в не зависимости от инстанса
	1 - использовать переданный через get запрос инстанс
	2 - все инстансы по очереди
&S...&E - блоки замены
Шаблон блоков
&S(id)*(ch)*(txt1)*(txt2)*(txt3)&E
id - филда с которого нужно брать значение
ch - тип данных {d - дата; s- строка}
txt1 - default ’екст автозамены при отсутствии значениЯ
txt2 - префикс (только если не default)
txt3 - постфикс (только если не default)
